# @workkit/notify-email

Email transport adapter for [`@workkit/notify`](../notify). Resend HTTP API + optional React Email rendering. Webhook signature verification (Svix-format), automatic opt-out on hard bounces and complaints.

## Install

```bash
bun add @workkit/notify-email @workkit/notify
# Optional, only if you use React Email components for templates
bun add @react-email/render
```

## Quick start

```ts
import { emailAdapter } from "@workkit/notify-email";
import { optOut } from "@workkit/notify";

const email = emailAdapter({
  apiKey: env.RESEND_API_KEY,
  from: "Reports <reports@entryexit.ai>",
  replyTo: "support@entryexit.ai",
  bucket: env.REPORTS,                               // optional, only if templates carry attachments
  webhook: { secret: env.RESEND_WEBHOOK_SECRET },
  autoOptOut: {
    enabled: true,
    hook: async (userId, channel, notificationId, reason) => {
      // The webhook only knows the email address; resolve to your userId here.
      const realUserId = await lookupUserIdByEmail(userId);
      if (realUserId) await optOut(env.DB, realUserId, channel, notificationId, reason);
    },
  },
  disableTrackingFor: ["pre-market-brief"],          // notification ids where open/click tracking is suppressed
});

// Plug into @workkit/notify
const notifyDeps = { adapters: { email }, /* ... */ };
```

## API

### `emailAdapter(options): Adapter<EmailPayload>`

- `apiKey` ŌĆö Resend API key.
- `from` ŌĆö RFC-5322 friendly form: `name@domain` or `Name <name@domain>`. Validated at construction.
- `replyTo` ŌĆö `string | string[]`.
- `apiUrl` ŌĆö override Resend API endpoint (testing).
- `bucket` ŌĆö `R2Bucket`-shape; required only when templates declare `attachments(payload)`.
- `attachments` ŌĆö `{ maxTotalBytes?: 40MB, concurrency?: 4 }`.
- `webhook.secret` ŌĆö Svix-format secret for `verifySignature`.
- `webhook.maxAgeMs` ŌĆö replay window. Default 5 min.
- `autoOptOut.enabled` ŌĆö default true; the hook decides what to write.
- `autoOptOut.hook` ŌĆö `(userId, "email", notificationId|null, reason) => Promise<void>`. **Note**: the userId passed in is the email address from the webhook payload ŌĆö Resend webhooks don't carry your internal id. Resolve to your real user id inside the hook.
- `disableTrackingFor` ŌĆö notification ids that should not be tracked.

### `renderEmail({ template, props?, text? })`

- `template: string` ŌĆö treated as ready-to-send HTML.
- `template: ReactElement` ŌĆö rendered via `@react-email/render` (optional peer; throws if not installed).
- `text` ŌĆö explicit plain-text body. If absent, generated by `htmlToText(html)`.

### `htmlToText(html)`

Strip script/style blocks, strip tags, decode the most common entities, collapse whitespace. Good enough for an automatic plain-text fallback; pass an explicit `text` for fidelity.

### Webhook helpers

- `parseResendEvents(rawBody)` ŌĆö maps `email.delivered/bounced/complained/opened/clicked` to `WebhookEvent[]`. Complaints surface as `bounced` (delivery-side rejection).
- `verifyResendSignature(req, secret)` ŌĆö validates `svix-signature` header (`v1,<base64>` HMAC-SHA256), enforces 5-min replay window.
- `isComplaint(rawEvent)`, `isHardBounce(rawEvent)` ŌĆö classifiers used by `autoOptOut`.

### Errors

- `FromDomainError` ŌĆö invalid `from` at construction.
- `AttachmentTooLargeError` ŌĆö payload exceeds the configured cap.
- `WebhookSignatureError` ŌĆö signature verification failed.

## Security defaults

- **Direct `fetch`** to Resend, no SDK ŌĆö small surface, easy to mock in tests.
- **Plain-text auto-generated** to avoid spam-classifier penalties on HTML-only emails.
- **Webhook signature required** when `webhook.secret` is supplied; helper enforces 5-min replay window.
- **Auto opt-out on hard bounce + complaint** (configurable; default on). Resend's webhook stream is signature-verified before opt-outs land.
- **From-domain validated at adapter init** (no surprise 403s in production).
- **Tracking off by default for notification ids in `disableTrackingFor`** ŌĆö financial/medical/etc.
- **Attachment cap** default 40MB; throws `AttachmentTooLargeError` rather than silently truncating.
- **R2 fetch parallelism bounded** (default 4) to keep Workers CPU budget manageable.

## Versioning

Follows the workkit Constitution ŌĆö single `src/index.ts` export, no cross-package imports outside declared peer deps. Changesets accompany every public API change.
